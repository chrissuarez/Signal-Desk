# Signal Desk - Opportunity Agent

Production-quality agent that finds job postings and business opportunities for you.

## Quick Start

1. `docker compose up -d`
2. `cd backend && npm install && npm run db:push && npm run dev`
3. `cd frontend && npm install && npm run dev`
